API-dokumentasjon

APIets URL er https://abakus-api-dot-sinuous-tine-156112.appspot.com/.
Bruk metodene /put, /search og /clear for å ta APIet i bruk, som beskrevet under.

/put

Lagrer en side i indexen deres, som dere kan søke i senere. 

      REQUEST
        
              curl -v -XPOST -H 'Content-Type: application/json' -d 
              '[{ "id": "1", 
                  "title": "Om oss", 
                  "contents": "Computas er et løsningshus basert i Norge", 
                  "url": "http://computas.com/om"
                }]' 

              http://localhost:5000/f61fbb45-b9bd-41ea-8b46-a817df01d1bb/put
        
        
        RESPONSE
        
              {
                "indexed_documents": 1
              }

/search

Henter ut sider i indexen som matcher søkestrengen. 

      REQUEST
        
          curl -v http://localhost:5000/f61fbb45-b9bd-41ea-8b46-a817df01d1bb/search  
          -H 'Content-Type: application/json' 
          -d '{"query":"bulk"}'
        
      
        RESPONSE
        
          {
            "index": "f61fbb45-b9bd-41ea-8b46-a817df01d1bb",
            "results": [
            {
              "contents": "Bulk indexing might work",
              "created_at": "2017-03-15T15:54:52.423918",
              "id": "2",
              "score": 0.35355338,
              "title": "Bulk indexing",
              "url": "http://theurl"
            },
            {
              "contents": "Bulk indexing might work. might overwrite too",
              "created_at": "2017-03-15T16:08:44.608036",
              "id": "3",
              "score": 0.35355338,
              "title": "Bulk indexing",
              "url": "http://theurl"
            }
            ],
            "total": 2
          }
